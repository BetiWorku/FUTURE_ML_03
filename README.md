# FUTURE_ML_03
#Task 3: Resume / Candidate Screening System
  Domain: Machine Learning  
  CIN ID: FIT/FEB26/ML5800  

 Project Overview
This project is an automated ML system designed to screen and rank resumes based on specific job descriptions. It helps HR teams quickly identify the most relevant candidates from a large pool of applicants using Natural Language Processing (NLP).

 Key Features
Resume Parsing: Automated cleaning and preprocessing of raw resume text.
Skill Extraction:Identifying key technical skills and experiences.
Candidate Ranking: Using **TF-IDF Vectorization** and **Cosine Similarity** to score candidates against job requirements.
Visualization:** Graphical representation of candidate scores for easy decision-making.

 Technologies Used
 Language:** Python
 Libraries:** Pandas, Scikit-learn, Matplotlib
 Tool: Jupyter Notebook

  Results
The model successfully ranks candidates with a Match Score.
- Top Match: Candidates with high overlap in keywords (Python, ML, NLP).
- Non-Match: Candidates with zero overlap are filtered out (Score: 0.00).
